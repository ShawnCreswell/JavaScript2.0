An Object is refernece datatype with key: value pairs.

An object in real life has characteristics and things it can do. For instance a car has a make, modle, number of wheels, color, etc..... These are the characteristics of the car. A car can also do drive, beep, etc. These are things a car can do. But in JS - we call these characteristics Properties

And we call things an object can do methods A method is just a function that belongs to an object
